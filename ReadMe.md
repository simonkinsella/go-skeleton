Simple Go Project Skeleton
==========================

A starting layout for new Go projects.


Installing Go (the easy way)
----------------------------

1. [https://go.dev/dl/]
2. Choose a *Featured download*, e.g "Apple macOS (ARM64)"
3. Download and run the installer
4. Open a new terminal
5. Type `go version`
6. If you see something like `go version go1.25.6 darwin/arm64`, you're done!
