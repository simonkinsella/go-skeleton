Simple Go Project Skeleton 🩻
=============================

A starting layout for new Go projects.


What you need
-------------
* Go
* Task
* An editor or IDE

See below for more info 👇


Installing Go (the easy way) 📦
-------------------------------

1. https://go.dev/dl/
2. Choose a **Featured download**, e.g `Apple macOS (ARM64)`
3. Download and run the installer
4. Open a new terminal 

5. Check:
```
> go version 
go1.25.6 darwin/arm64
```

Installing Taskfile 📜
----------------------
[Task](https://taskfile.dev/) is used in this repo as a modern alternative to `Make`.  

To install `Task`: https://taskfile.dev/docs/installation

Check:
``` 
> task --version
3.50.0
```

Visual Studio Code
------------------
VS Code is a good no-cost choice: https://code.visualstudio.com/download

Installing the "Go for Visual Studio Code" extension is **Highly Recommended!** 📣
